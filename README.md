## <a name="overview"></a>Overview ##

Lightfielder Extras is an open-source set of post-production actions, macros, and small tools that help make volumetric production easier by providing a set of consistent tools that run inside of the leading DCC apps. Look in the individual folders in this repo for each of the sets of Lightfielder tools and their supporting documentation.
